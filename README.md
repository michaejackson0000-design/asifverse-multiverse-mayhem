# ASIFVERSE: MULTIVERSE MAYHEM

## Game Overview
ASIFVERSE: MULTIVERSE MAYHEM is a polished, original action arcade game built in Godot 4. It delivers rapid-fire mini-games, neon cyberpunk visuals, satisfying feedback, and a modular architecture designed for future expansion.

## Story
The player is a neon courier moving through unstable portals across the multiverse. Each portal drops them into a new challenge where reflexes, timing, and composure determine survival.

## Gameplay
- Fast-paced arcade action with short mini-games lasting 3–6 seconds.
- Randomized rounds with escalating difficulty.
- Boss battles every 10 rounds.
- Combo, score, streak, lives, and multiplier systems.
- Save system for high scores, settings, achievements, and progress.

## Features
- Modular autoload managers for gameplay, audio, scene transitions, save, scoring, settings, achievements, and effects.
- Reusable mini-game base class for future content creation.
- Responsive UI flow from splash screen to main menu, settings, credits, gameplay, and game over.
- Cyberpunk pixel-art inspired visuals and juicy feedback hooks.

## Controls
- Space / Enter: Confirm or interact.
- Mouse movement: Navigate and interact.
- A / D: Movement in custom gameplay hooks.

## Installation
1. Install Godot 4.
2. Open the project folder in Godot.
3. Press F5 to run the game.

## How to Build

- Export for your target platform using the standard Godot export workflow.
